# homebrew-tap

Homebrew tap for Austin Cherry's projects.

## Installation

```bash
brew tap acmacalister/tap
```

## Available Formulas

### swg

Secure Web Gateway - HTTPS MITM proxy for content filtering.

```bash
brew install acmacalister/tap/swg
```

Or after tapping:

```bash
brew install swg
```

## Usage

After installation, run:

```bash
swg --help
```
